<CSFrame>
    <CSTitle>First-Order Linear ODEs</CSTitle>

    Write the ODE in the form:

    <DisplayLatex code="
        \frac{\diff{y}}{\diff{x}} + f(x) \, y = g(x)
    " />

    First, define the *integrating factor* <Eq code="h" />:

    <DisplayLatex code="
        h(x) = e^{\int{f(x) \,\diff{x}}}
        \Exn{
            \qquad \Longrightarrow \qquad
            h'(x) = f(x) \, h(x)
        }
    " />

    Multiply the ODE by <Eq code="h" /> and undo the product rule:

    <DisplayLatex code="
        h(x) \, \frac{\diff{y}}{\diff{x}} + h(x) \, f(x) \, y = g(x) \, h(x) \\
        \frac{\diff{}}{\diff{x}} \parens{h(x) \, y} = g(x) \, h(x)
    " />

    To solve, integrate both sides:

    <DisplayLatex code="
        h(x) \, y = \int{g(x) \, h(x) \,\diff{x}}
    " />
</CSFrame>

<CSFrame>
    <CSTitle>Exact ODEs</CSTitle>

    Write the ODE in the form:

    <DisplayLatex code="
        F(x, y) + G(x, y) \frac{\diff{y}}{\diff{x}} = 0
    " />

    First prove the ODE is *exact* by showing:

    <DisplayLatex code="
        \frac{\partial{F}}{\partial{y}}
        = \frac{\partial{G}}{\partial{x}}
        \Exn{
            {} = \underbrace{
                \frac{\partial^2{H}}{\partial{x} \, \partial{y}}
                = \frac{\partial^2{H}}{\partial{y} \, \partial{x}}
            }_{\substack{\text{equality of}\\\text{mixed partials}}}
        }
    " />

    Thus, our ODE can be rewritten:

    <DisplayLatex code="
        H_x(x, y) + H_y(x, y) \frac{\diff{y}}{\diff{x}} = 0
    " />

    This can be simplified by undoing the chain rule:

    <GatherLatex code="
        \frac{\partial{H}}{\partial{x}} \frac{\diff{x}}{\diff{x}}
        + \frac{\partial{H}}{\partial{y}} \frac{\diff{y}}{\diff{x}}
        = 0 \\
        \frac{\diff{}}{\diff{x}} \parens{H \parens{x, y(x)}} = 0
    " />

    Therefore, the solution is:

    <DisplayLatex code="
        H(x, y) = C
    " />

    such that:

    <AlignatLatex
        eqcols={3}
        code="
            H(x, y) &= \int{H_x(x, y) \,\partial{x}} &&= H_1(x, y) &&+ C_1(y) \\
            H(x, y) &= \int{H_y(x, y) \,\partial{y}} &&= H_2(x, y) &&+ C_2(x)
        "
    />
</CSFrame>

<CSFrame>
    <CSTitle>Trigonometric Fourier Series</CSTitle>

    For any *periodic function* <Eq code="f \parens{t} = f \parens{t + n T}" />:

    <AlignLatex code="
        f \parens{t}
        &= a_o
        + \sum_{n=1}^\infty \parens{
            a_n \cos n \omega_0 t + b_n \sin n \omega_0 t
        }
        \\
        &= a_o
        + \sum_{n=1}^\infty {
            A_n \cos \parens{n \omega_0 t + \phi_n}
        }
        \qquad\quad
        \begin{array}{l}
            \text{\footnotesize{}(Amplitude-} \\
            \text{\footnotesize{}Phase Form)} \\
        \end{array}
        %\htmlStyle{width: 5em;}{\text{\footnotesize{}(Amplitude-Phase Form)}}
    " />

    <GatherLatex code="
        a_0
        = \frac{1}{T} \int_0^T f\parens{t} \,\diff{t} = \text{average value of $f$}
        \\
        a_n
        = \frac{2}{T} \int_0^T f\parens{t} \cos n \omega_0 t \,\diff{t}
        \\
        b_n
        = \frac{2}{T} \int_0^T f\parens{t} \sin n \omega_0 t \,\diff{t}
        \\
        A_n = \sqrt{a_n^2 + b_n^2}
        \qquad\quad
        \phi_n = - \tan^{-1} \frac{b_n}{a_n}
    " />

    Fundamental angular frequency:

    <DisplayLatex code="
        \omega_0 = \frac{2 \pi}{T}
    " />

    Each element of the sum is called a *harmonic*.

    Dirichlet conditions (for a convergent Fourier series):

    - <Eq code="f" /> must be single-valued everywhere.
    - <Eq code="f" /> must have a finite number of finite discontinuities, maxima, and minima in any one period.
    - <Eq code="\int_{t_0}^{t_0+T} \abs{f\parens{t}} \,\diff{t} < \infty" /> for any <Eq code="t_0" />.
</CSFrame>

<CSFrame>
    <CSTitle>Exponential Fourier Series</CSTitle>

    Alternative to the *trigonometric Fourier series*:

    <DisplayLatex code="
        f\parens{t}
        = \sum_{n = -\infty}^\infty c_n e^{i n \omega_0 t}
        = c_0 + \sum_{n = 1}^\infty \parens{
            c_n e^{i n \omega_0 t} + c_{-n} e^{i n \omega_0 t}
        }
    " />
    <DisplayLatex code="
        c_0 = a_0
        \qquad\quad
        c_n = \frac{1}{T} \int_0^T f\parens{t} \, e^{-i n \omega_0 t} \,\diff{t}
        , \quad \forall n \ne 0
    " />

    Related by:

    <DisplayLatex code="
        A_n \phase{\phi_n} = a_n - i b_n = 2 c_n, \quad \forall n \ne 0
    " />
</CSFrame>

    <CSTitle>Fourier Series: Useful Integral Identities</CSTitle>
    <DisplayLatex code="
        n, m \in \myIntegerSet
        \\[0.4em]

        \\

        , \qquad \parens{n \ne m}
        \\
        \int_0^T \sin^2{n \omega_0 t} \,\diff{t} = \frac{T}{2}
        \\
        \int_0^T \cos^2{n \omega_0 t} \,\diff{t} = \frac{T}{2}
        \\[1.8em]
        \int t \cos{a t} \,\diff{t}
        = \frac{1}{a^2} \cos{a t} + \frac{1}{a} t \sin{a t} + C
        \\
        \int t \sin{a t} \,\diff{t}
        = \frac{1}{a^2} \sin{a t} - \frac{1}{a} t \cos{a t} + C
    " />
</CSFrame>

<CSFrame>
    <CSTitle>Fourier Transform</CSTitle>

    For function <Eq code="f" /> *(not necessarily periodic)* which we interpret as a periodic function with *infinite period*:

    <GatherLatex code="
        F\parens{\omega}
        = \FourierTransform\brackets{f\parens{t}}
        = \int_{-\infty}^\infty f\parens{t} \, e^{-i \omega t} \,\diff{t}
        \\
        f\parens{t}
        = \FourierTransform^{-1}\brackets{F\parens{\omega}}
        = \frac{1}{2 \pi}
        \int_{-\infty}^\infty F\parens{\omega} \, e^{i \omega t} \,\diff{\omega}
    " />

    The Fourier transform <Eq code="F" /> exists when the Fourier integral converges.
</CSFrame>

    <DisplayLatex code="

    <CSTitle>Laplace Transform</CSTitle>

    For function <Eq code="f" /> and some complex variable <Eq code="s = \sigma + i \omega" />:

    <DisplayLatex code="
        \LaplaceTransform\brackets{f\parens{t}}
        = F\parens{s}
        = \int_{0^-}^\infty f\parens{t} e^{-s t} \,\diff{t}
    " />
</CSFrame>

<CSFrame>
    <CSTitle>Laplace Transform Properties</CSTitle>

    Linearity:
    <DisplayLatex code="
        \LaplaceTransform\brackets{a_1 f_1(t) + a_2 f_2(t)}
        = a_1 F_1\parens{s} + a_2 F_2\parens{s}
    " />

    Scaling:
    <DisplayLatex code="
        \LaplaceTransform\brackets{f\parens{a t}}
        = \frac{1}{a} F\parens{\frac{s}{a}}
    " />

    Time Shifting:
    <DisplayLatex code="
        \LaplaceTransform\brackets{f\parens{t - a} \, u\parens{t - a}}
        = e^{-a s} F\parens{s}
    " />

    Frequency Shifting/Translation:
    <DisplayLatex code="
        \LaplaceTransform\brackets{e^{-a t} f\parens{t} \, u\parens{t}}
        = F\parens{s + a}
    " />

    Time Differentiation:
    <GatherLatex code="
        \LaplaceTransform\brackets{f'\parens{t}}
        = s F\parens{s} - f\parens{0^-}
        \\
        \LaplaceTransform\brackets{f''\parens{t}}
        = s^2 F\parens{s} - s f\parens{0^-} - f'\parens{0^-}
    " />

    <AlignLatex code="
        \LaplaceTransform\brackets{f^{\parens{n}}\parens{t}}
        &= s^n F\parens{s}
        - s^{n-1} f\parens{0^-}
        - s^{n-2} f'\parens{0^-}
        \\
        &\phantom{=} - \cdots
        - s^0 f^{\parens{n-1}}\parens{0^-}
    " />

    Time Integration:
    <DisplayLatex code="
        \LaplaceTransform\brackets{\int_0^t f\parens{x} \,\diff{x}}
        = \frac{1}{s} F\parens{s}
    " />

    Frequency Differentiation:
    <DisplayLatex code="
        \LaplaceTransform\brackets{t^n f\parens{t}}
        = \parens{-1}^n F^{\parens{n}}\parens{s}
    " />

    Time Periodicity:
