- **Motherboard: Asrock X870E Taichi** (BIOS version 3.20)

## BIOS Settings Locations

- BIOS > OC Tweaker
    - (1) **DRAM Profile Configuration > DRAM Profile Setting**
- BIOS > Advanced > AMD Overclocking > Precision Boost Overdrive
    - (2) **Precision Boost Overdrive**
    - (3) **PBO Limits**
    - (4.1) **CPU Boost Clock Override**
    - (4.2) **Max CPU Boost Clock Override(+)**
    - Curve Optimizer
        - (5.1) **Curve Optimizer**
        - (5.2) **All Core Curve Optimizer Sign**
        - (5.3) **All Core Curve Optimizer Magnitude**
    - Curve Shaper
        - TODO: (6)
