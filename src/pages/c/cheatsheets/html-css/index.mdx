## Flexbox

### Base Example

```html
<div class="a">
    <div>1</div>
    <div>2</div>
    <div>3</div>
    <div>4</div>
    <div>5</div>
</div>
```

<SpoilerBlock buttonLabel="Base CSS">

/* This is all purely styling unrelated to flexbox. */

    width: 20em;
}

div.a > * {
    border: 0.2em solid blue;
    background: #faa;
    width: 2em;
    height: 2em;

}
