## Text

```css
{
    font-weight: bold;
    font-weight: normal;
    font-weight: lighter;
    font-weight: 400; /* numeric values are in [1,1000] */

    font-family: "Courier New", monospace; /* left is highest priority */
}
```

## Flexbox

### Base Example

```html
<div class="a">
    <div>1</div>
    <div>2</div>
    <div>3</div>
    <div>4</div>
    <div>5</div>
</div>
```

```css
/* This is all purely styling unrelated to flexbox. */
/* I'm just establishing a baseline with this. */
div.a {
    border: 0.2em solid red;
    background: white;
}

div.a > * {
    border: 0.2em solid blue;
    background: #faa;
    width: 2em;
    height: 2em;
    text-align: center;
}
```

### Some Alternative Attributes

```css
{
    column-gap: 0.5em; /* alternative to 'gap' */
    row-gap: 0.5em;    /* alternative to 'gap' */
}
```
